## 📦 Installation & Setup

### 👉 Run Frontend
```bash
npm install
npm run dev
```

### 👉 Run Backend
```bash
npm install
npm run dev
```
###  👉👉👉👉👉👉👉 .env
```bash

MONGO_DB =  your mongodb url
JWT_KEY = your jwt key
PORT = 8000
```
